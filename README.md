# report-discord
## Описание
Создан для оценки пользователей на серверах дискорд.
Не будет работать если не укажете ID канала в котором будет отправляться информация о репортах 

## Библиотеки
Требуется скачать dotenv и discord.py
```
pip install discord.py
pip install dotenv-python
```

## Команды
`-rep <Пинг юзера> <Причина>` - Отрицательная оценка пользователя с обязательной написанием причиной. Если вы хотите написать больше в "Причине", то просто добавьте ковычки("")

`+rep <Пинг юзера> <Причина>` - Положительная оценка пользователя с обязательной написанием причиной, соответствено.

`.stats <Пинг юзера>` - Статистика сколько отправлено положительных и отрицательных оценок.

`.comment <Пинг юзера>` - Отправляет какие комментарие были отправлены вам

`.clear <Количество сообшение <= 100>` - Чистка текстовых каналов. За одну команду можно удалить 100 сообшение, не более.
